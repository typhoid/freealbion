This file contains the names of all the items within the game. They are translated into English, German and French.

Each record is 60 bytes long and there are 462 records.

| **Offset** | **Length** | **Description** |
|:-----------|:-----------|:----------------|
|0           |20          |German text      |
|20          |20          |English text     |
|40          |20          |French text      |
